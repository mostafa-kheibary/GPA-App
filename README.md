# Simple React app to calculate GPA(grade point avrage)

## you can ...
- add or delete lesson
- see the result in simbolic and number form


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
